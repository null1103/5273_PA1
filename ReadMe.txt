Building and Using the Program
------------------------------

1. Run 'make server' and 'make client'
2. Run './client [server_ip][port_no]' and './server[port_no]'

Commands
PUTF - Put a file
GETF - Get a file
LIST - List all files
EXIT - Exit the server

Above commands have been implemented in the programs where client sends one of the above commands to the server. 
Based on the command received it works on the files in the current directory or the server exits.
Any other command not recognized. The system works to transfer files(or data) upto 5 KB.
